# Gurren Drum Kit (Tema TTGL)

Versión temática del reto **JavaScript Drum Kit** (JavaScript30) inspirada en *Tengen Toppa Gurren Lagann*.
Reproduce sonidos al presionar teclas y aplica una estética inspirada en la imagen del anime.

## Estructura
- `index.html` — interfaz principal
- `style.css` — estilos con tema TTGL (fondo, colores, animaciones)
- `script.js` — lógica para reproducir sonidos y animar teclas
- `assets/TTGL_scaled.jpg` — imagen de fondo 
- `sounds/` — carpeta para colocar archivos .wav/.mp3

## Cómo usar
1. Clona el repositorio.
2. Coloca tus sonidos en `sounds/` (con los mismos nombres).
3. Abre `index.html` en un navegador moderno.

## Deploy
Funciona desde GitHub Pages, Vercel o Netlify. Asegúrate de que la carpeta `assets/` y `sounds/` se publiquen.
